# United Loading
一个显示不确定加载进度条的窗口

## 使用方法
1. 安装所有所需依赖
```bash
pip install -r requirements.txt
```

2. 安装 `HARMONYOS_SANS_SC_BLACK.TTF` 字体
3. 在终端中输入以下命令

```bash
"pythonw.exe" "united_ui.py" [加载文字]
```

### 示例
```bash
"pythonw.exe" "united_ui.py" 正在烧烤
```
